apps and sites

  • Bloodborne Ally (React app)

    Bloodborne Ally is a GraphQL API, built with Slicknode. The database contains hand-written entries for every loot item, boss, NPC hunter, and each "quest" event that connects them. The API is highly flexible and be used for complex relational queries. The React app, using Apollo to connect to the API, is a simple, but growing, UI for displaying the API docs and examples.
