apps and sites

  • Bloodborne Ally App

    Bloodborne Ally is a GraphQL API, built with Slicknode. The database contains hand-written entries for every loot item, boss, NPC hunter, and each quest event that connects them. The API is highly flexible and be used for complex relational queries. This React app, using Apollo to connect to the API, is a simple but growing UI for displaying the API docs and examples.

  • Things above the Earth

    React app to track the International Space Station in real time. Watch orbital path draw over Earth in multiple different projections powered by d3. Features coming soon: predict ISS flyovers, moon shadow tracking, panning & zooming, graticule display, tooltips, unfolding educational popovers

  • data.world

    UI development and maintenance on a complex React & Redux app for collaborative data sharing and analysis.

  • austincodingacademy.com

    Built new site for Austin Coding Academy with Jekyll and AMP. Multi-page state site developed from scratch and through multiple major content updates.
