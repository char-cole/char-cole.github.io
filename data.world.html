data.world | UI development in React

Link to data.world

data.world is a React app written in TypeScript. Charles has been working a contract position at data.world since May, 2019, developing UI features.

A key feature developed by Charles is something called derivative datasets. On data.world, data files are stored in datasets. Queries can be written in SQL or SPARQL against those files, and the results of the query can be saved as new files to the working dataset or to another dataset.

Derivative datasets were a commonly requested feature that would allow the user to create a new dataset to which the query results would be saved without leaving their workflow. Charles implemented this capability through a refactoring of the "Save Results Tables" modal into a branching component that determines the state of the user's current workspace and the user's access levels to make appropriate, sequenced calls to the API.

By entering the derivative dataset creation flow, the modal stores information about the file that will be created, then renders the dataset creation form and stores all of the inputs given to it. It then creates the new dataset and determines whether or not it should create a connection between the new dataset and the currently active one. Along the way the modal indicates through UI elements to the user what that process will be.
